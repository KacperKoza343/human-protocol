## Getting Started

### Prerequisites

- [Node.js minimal v18](https://nodejs.org/en)
- [pnpm](https://pnpm.io/installation)

### Install Packages

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```

### Linting and Formatting

```sh
pnpm lint
```
